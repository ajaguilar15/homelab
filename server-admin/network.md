## Networking

### Show Network Interfaces

```bash
ip a
```

Displays:
- Network interfaces
- IPv4 addresses
- IPv6 addresses
- Interface states
- MTU information

### Display IP Address

```bash
hostname -I
```

Prints the current assigned IP addresses.

### Test Network Connectivity

```bash
ping google.com
```

Used to:
- Verify internet connectivity
- Test DNS resolution
- Measure latency
- Diagnose networking issues

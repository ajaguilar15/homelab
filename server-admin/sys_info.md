# System Information

## Kernel and OS Information

```bash
uname -a
```

Displays:
- Kernel version
- System architecture
- Hostname
- Build information

## System Overview

```bash
fastfetch
```

Displays:
- Operating system
- CPU
- Memory usage
- Host platform
- Shell information
- Kernel version

Install:

```bash
sudo apt install fastfetch
```

# Package Management

## Update Package Lists

```bash
sudo apt update
```

Refreshes repository metadata.

## Upgrade Installed Packages

```bash
sudo apt upgrade
```

Updates installed software packages.

## Install a Package

```bash
sudo apt install <package>
```

Example:

```bash
sudo apt install fastfetch
```

## Remove a Package

```bash
sudo apt remove <package>
```

Example:

```bash
sudo apt remove cups-client
```
